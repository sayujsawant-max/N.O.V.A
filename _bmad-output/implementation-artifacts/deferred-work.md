- **`BrainPort.delete_matching(target: str)` — opaque-reference privacy invariant unencoded.** Docstring requires `target` to be opaque (e.g., `"mode 'opaque'"`), never a raw app name. The `str` type cannot carry the "never raw" constraint; mypy accepts a window-title string. **Target: Story 5.2 (Selective Forget).** Wrap in `OpaqueTarget = NewType("OpaqueTarget", str)` at minimum; ideally a dataclass with a validator.
- **`BriefingAggregate` triple-nullable with no valid-state matrix.** `last_session: SessionSummary | None`, `last_snapshot: WorkspaceSnapshot | None`, `last_seed: str | None` — 2³ = 8 possible states; docstring doesn't document which are valid. **Target: Story 3.2 (BriefingAggregate population).** Either document the matrix or model as a tagged union (FirstRun / WarmResume / etc.).
- **`WindowContext.is_opaque` invariant unenforced.** Docstring promises `is_opaque=True ⇒ all three identity fields are None`, but `WindowContext(app_name="code.exe", is_opaque=True, ...)` constructs cleanly. **Target: Story 4.2 (Exclusion boundary at capture).** Add `__post_init__` validator asserting the implication.
- **`ActionResult` tri-state `success`/`reason` has no invariant check.** Docstring says "`reason` carries failure reason when `success=False` and is `None` on success", but `ActionResult(success=True, reason="failed")` type-checks. **Target: Story 3.6 (Hands actions).** Add `__post_init__` validator OR switch to a tagged-union shape (`Success | Failure`).
- **Port docstrings cite absolute `architecture.md:NNN` line numbers.** Pervasive across 8 port files + 5 model files; any architecture.md edit shifts these silently. Today's cites are accurate. **Target: low-priority cleanup; revisit only if cites actually go stale.** If cleaning, replace with section-heading citations.
- **`test_ports_only_import_from_system_models_modules` accepts `segments[3] == "models"` but allows deeper nesting.** A future `nova.systems.brain.models.internal` would pass; today no such nesting exists. **Target: speculative — assert `len(segments) == 4` if the project ever introduces a nested `models/` package.**
